# Reverse Engineering & Malware Analysis Portfolio

Данный репозиторий представляет собой структурированный архив моих проектов в области обратной разработки (Reverse Engineering), анализа вредоносного ПО и исследования прошивок. 

Цель репозитория — демонстрация практических навыков статического и динамического анализа, обхода механизмов защиты и понимания низкоуровневой архитектуры программного обеспечения.

## Структура репозитория

Проекты разделены на директории в зависимости от домена исследования и уровня сложности.

### 1. Malware Analysis
Основная директория, содержащая отчеты по анализу исполняемых файлов. Задачи классифицированы по системе тиров (Tiers) для отражения прогрессии сложности:

* **Tier 0 (Foundation)**
  Базовые проекты. Включает в себя реверс-инжиниринг начальных Crackmes (Dfct 1.0 - 2.0), самописных скриптов и программ с открытой логикой.
  *Фокус: базовый статический и динамический анализ, патчинг, поиск строк, восстановление простых алгоритмов.*

* **Tier 1 (Intermediate)**
  Проекты среднего уровня. Сюда входит анализ реального, но технически простого вредоносного ПО (Malware), а также Crackmes повышенной сложности (Dfct 2.1 - 4.0).
  *Фокус: обход базового Anti-Debug, распаковка простых алгоритмов, анализ сетевой активности малвари, работа с упаковщиками.*

* **Tier 2 (Advanced)**
  Сложные проекты. Включает в себя анализ продвинутого вредоносного ПО, защищенных коммерческих протекторов и нестандартных Crackmes (zero quality / high difficulty).
  *Фокус: анализ обфусцированного кода, восстановление кастомной криптографии, работа с продвинутыми механизмами анти-анализа.*

### 2. Firmware & Embedded
Директория для статических исследований прошивок и встраиваемых систем. 
На данный момент раздел содержит завершенный анализ модуля **CALEA** маршрутизаторов Microtik. Подробный разбор архитектуры и принципов работы модуля находится в соответствующей директории.

## Инструментарий

Для проведения исследований используется следующий стек технологий и утилит:

* **Disassemblers & Decompilers:** IDA Pro, Ghidra, dnSpy (.NET)
* **Dynamic Analysis (Debuggers):** x64dbg / x32dbg
* **PE Analysis & Hex Editors:** PE-bear, HxD
* **System & Behavior Monitoring:** 
  * Process Hacker
  * Sysinternals Suite (Process Monitor / Procmon, TCPView)
* **Network Analysis:** Wireshark
* **Languages:** C/C++, Python, Assembly (x86/x64)
